# Audiencias · Creative Center

Herramienta interna para cruzar una base de contactos con ventas, crear segmentos comerciales y exportarlos a CSV.

## Privacidad

Los archivos se leen y procesan exclusivamente en el navegador. La aplicación no tiene base de datos, backend ni almacenamiento persistente.

## Uso local

1. Instalá las dependencias con `pnpm install`.
2. Iniciá la app con `pnpm dev`.
3. Abrí la dirección local indicada en la terminal.

## Publicación

Es una aplicación estática compatible con Cloudflare Pages, Vercel, Netlify y GitHub Pages.
